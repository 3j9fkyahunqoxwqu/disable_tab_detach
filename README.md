Disable Tab Detach
==================

Firefox add-on that returns a tab to its original window and position on detach


## Keyboard Shortcuts

| Command                | Windows / Linux  | macOS         |
| ---------------------- | ---------------- | ------------- |
| Move tab to new window | `Ctrl`+`Alt`+`N` | ⌥⌘N           |
| Pause extension        | `Ctrl`+`Alt`+`P` | ⌥⌘P           |
